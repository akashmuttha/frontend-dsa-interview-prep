# Morning Study System

Work usually runs from 11:30 AM to 9:00 PM, so morning is the main study window.

## Default Morning

```text
DSA:       60-75 minutes
Frontend: 45-60 minutes
Recall:   10-15 minutes
```

## Priority

DSA comes first because it needs fresh thinking and pattern recall.

Frontend comes second because it compounds through repeated small code outputs.

## Bad Morning Minimum

```text
DSA:       30 minutes
Frontend: 20 minutes
```

The goal is to keep the chain alive without depending on perfect days.

## Night Rule

After work, serious study is optional. Recovery is part of the system.
