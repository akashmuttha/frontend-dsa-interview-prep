# Running Sum of 1D Array

## Link

https://leetcode.com/problems/running-sum-of-1d-array/

## Statement

Given an array `nums`, return the running sum where each index contains the sum of all values from the start of the array up to that index.

## Example

```text
Input: nums = [1,2,3,4]
Output: [1,3,6,10]
```

## Pattern

Prefix sum.
