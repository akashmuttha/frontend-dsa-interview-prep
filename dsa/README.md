# DSA Track

DSA is practiced as pattern recognition, not question counting.

Primary solving language: Python.

Selected problems may also include JavaScript or TypeScript rewrites for frontend interview fluency.

## Roadmap

See `roadmap.md` for the full DSA flow used in this repo.

High-level order:

1. Foundations: Big O, arrays, hash maps, prefix sum, two pointers, sliding window, binary search
2. Core data structures: linked lists, stacks, queues, trees, heaps, graphs
3. Advanced problem solving: backtracking, dynamic programming, mixed patterns
4. Interview readiness: timed practice, mock interviews, pattern revision

## Folder Map
