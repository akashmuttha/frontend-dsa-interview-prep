# DSA Track

DSA is practiced as pattern recognition, not question counting.

## Order

1. Foundations: Big O, arrays, hash maps, two pointers, sliding window
2. Data structures: linked list, stack, queue, trees, heaps, graphs
3. Advanced problem solving: backtracking and dynamic programming
4. Interview readiness: timed practice, mocks, revision

## Problem Folder Format

```text
problem-name/
  problem.md
  solution.ts
  notes.md
```
