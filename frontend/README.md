# Frontend Track

Frontend preparation is focused on becoming a stronger product frontend engineer, not only passing trivia interviews.

## Order

1. JavaScript fundamentals
2. TypeScript fundamentals
3. React hooks and component patterns
4. Machine coding practice
5. Portfolio-quality projects

## Practice Standard

Each concept should include:

- Notes in your own words
- A small coded example
- One interview-style explanation

Each project should include:

- README
- Runnable code
- Screenshots or deployment link when ready
- Tradeoffs and improvements
