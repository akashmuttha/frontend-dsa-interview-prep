# JavaScript Track

JavaScript practice is recall-first: implement or explain before rewatching.

## Focus Areas

- Values, types and variables
- Functions, arrays and objects
- References, copying and immutability
- Scope, closures, hoisting and TDZ
- `this`, arrow functions and explicit binding
- Constructors, prototypes and classes
- Promises, event loop and async/await
- DOM events, delegation, storage and modules
- Browser, networking and security basics
- Utility implementations and output questions

## Practice Outputs

- `concepts/` for explanations and short examples
- `practice/` for utility implementations and output-question files
